# QuicklyTanking
## 本程序作为2022年西邮3g实验室安卓组冬季项目。
### 整体使用组件化， 使用ARouter路由， java为主， Kotlin为辅（可能有C， C#， C++）
### Room与文件存储作为本地数据处理， mvvm架构/mvc架构，Retrofit负责网络处理
### Glide负责图像处理，Hilt与Dagger负责依赖注入, 使用ViewModel与仓库模式来提供数据
